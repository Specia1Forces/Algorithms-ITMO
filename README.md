# Algorithms-ITMO
Решал задачи по следующим темам:

* бинарный поиск;
* сортировки;
* системы непересекающихся множеств (DSU);
* очереди и кучи;
* динамическое программирование;
* алгоритм Флойда — Уоршелла;
* алгоритм Форда — Беллмана;
* поиск кратчайшего пути;
* поиск отрицательного цикла;
* компоненты связности;
* проверка графа на наличие цикла;
* двудольная раскраска графа;
* конденсация графа;
* мосты в графах;
* построение остовного дерева.
